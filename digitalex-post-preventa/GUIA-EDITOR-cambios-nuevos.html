Animación de los botones nuevos · regla global

Aplica a los botones 1, 3 y 10

Los tres botones nuevos (banner sticky superior, sección AMPLIAMOS EL AFORO y sticky bottom mobile) deben tener una animación de pulso suave continuo. El movimiento llama la atención del usuario sin distraer y aumenta la tasa de click sobre botones estáticos.

@keyframes dgx-pulse {
  0%, 100% {
    transform: scale(1);
    box-shadow: 0 0 0 0 rgba(31, 207, 211, 0.55);
  }
  50% {
    transform: scale(1.05);
    box-shadow: 0 0 0 10px rgba(31, 207, 211, 0);
  }
}

/* Aplicar a cada botón nuevo: */
.boton-nuevo {
  animation: dgx-pulse 1.8s ease-in-out infinite;
  transition: transform 0.2s ease;
}
.boton-nuevo:hover {
  transform: scale(1.06);
  animation-play-state: paused;
}
Comportamiento:
  • El botón crece y decrece sutilmente (escala 1 → 1.05 → 1) cada 1.8 segundos
  • Una sombra turquesa se expande y desvanece alrededor del botón en cada pulso
  • Al pasar el mouse encima (hover), la animación se pausa y el botón crece a 1.06 (estado activo)
  • El efecto es discreto: llama la atención, no marea
